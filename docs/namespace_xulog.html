◆ AsyncType

enum class Xulog::AsyncType
strong

异步工作器类型

定义了异步工作器的两种类型：

  • ASYNC_SAFE: 当缓冲区满时，阻塞生产者
  • ASYNC_UNSAFE: 不考虑资源，无限制扩容，适用于性能测试
枚举值
ASYNC_SAFE 

缓冲区满则阻塞

ASYNC_UNSAFE 

不考虑资源，无限扩容，性能测试
